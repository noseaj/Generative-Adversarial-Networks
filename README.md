# Generative-Adversarial-Networks

(22/07 ~ 23/02) 광운대학교 산학연계 프로젝트 - Team 사미용두

## Introduction
Apply Generative Adversarial Nets to generating leak data.

## 주요 기능
1. 리크 데이터 생성
>* GAN 모델로 랜덤 생성

2. 불량 이미지 생성
>* 생성 위치 랜덤 지정
>* 생성 위치 사용자 지정
>* 여러개 리크 생성

3. 불량 이미지 다운로드
